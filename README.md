# NEB GPA Calculator

This is a small project I made for NEB Grade 12 students in Nepal. The main idea was to make GPA calculation simple, fast, and easy to use for both Science and Management streams.

I built it because a lot of students just want a quick way to check their GPA without dealing with a complicated interface. So I kept the design clean, mobile-friendly, and easy to understand.

## What it does

- Calculates GPA for NEB Science and Management streams
- Handles subject-wise theory and practical grades
- Shows a marksheet-style result
- Lets users download the result as a PDF
- Works well on mobile and desktop

## Pages in this project

- `index.html` – Home page
- `science/` – Science GPA calculator

- `privacy.html` – Privacy Policy
- `terms.html` – Terms of Service
